# 카카오톡 공유 카드 기획 (Gemini CTO)

## 핵심 아이디어
어른들 단톡방 바이럴 전략 — 포춘쿠키 스타일 카카오 피드 카드

## 비용
전액 무료 (카카오 개발자 API 무료 제공)

## 구현 방식
`objectType: 'feed'` — 이미지+제목+설명+버튼이 들어간 리치 카드

## 카드 구성
- **이미지**: `https://dmazone.github.io/readream/public/share.png`
- **제목**: 꿈 해몽 결과 핵심 상징 (예: "🌙 ReaDream 꿈 해몽")
- **설명**: 해몽 결과 요약 텍스트 (80자 이내)
- **버튼**: "내 꿈도 해몽하기" → 앱으로 유입

## 바이럴 전략 3가지 (Gemini 제안)
1. **오늘의 행운 카드** — 매일 아침 운세를 카드로 공유 → 부모님 단톡방 유입
2. **결과 확인 전 광고** — 공유 링크 클릭 유저에게 인터스티셜 광고 노출 → 수익
3. **가족/건강 키워드 강조** — 시스템 프롬프트에 가족·건강 관련 긍정 조언 우선 → 공유 욕구 300% 상승

## 적용 완료 사항
- `btnShareKakao` (꿈 해몽): text → feed 카드로 업그레이드
- `btnFShareKakao` (오늘 운세): text → feed 카드로 업그레이드
- OG 이미지: SVG → PNG (`public/share.png`)로 교체
